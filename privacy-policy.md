Privacy Policy for AccelPro mobile App

Last updated: 2025-01-27

AccelPro (“the App”) is designed for acceleration data acquisition.
This Privacy Policy explains how the App handles user and device data.

1. Data We Collect

AccelPro collects acceleration sensor data from the user’s device only while the user is actively using the App.

Collected sensor data includes:

X-axis acceleration

Y-axis acceleration

Z-axis acceleration

No personal information is collected.

2. How We Use the Data

Acceleration data is used exclusively for:

Displaying acceleration components on charts

Allowing the user to store the data locally

Allowing the user to share the data manually (e.g., through email or file sharing)

All processing happens locally on the user’s device.

3. Data Storage

If the user chooses to save or export the data:

Data is stored only on the user’s device

Users have full control over saving, exporting, and deleting their data

No data is stored on external servers

4. Data Sharing

AccelPro does not send any information to:

Third parties

External servers

Analytics providers

Advertising networks

The App will only share data if the user chooses to share it manually, such as through email or messaging apps.

5. Third-Party Services

AccelPro does not use:

Third-party analytics

Third-party ads

Tracking tools

Cloud services

No automatic data transmission takes place.

6. Children’s Privacy

AccelPro is not directed to children under 13.
The App does not knowingly collect personal information from children.

7. Security

All data processing occurs locally on the device.
Since no data is transmitted or stored externally, there are no external security risks related to remote servers.

8. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time.
The latest version will always be available on this page.
